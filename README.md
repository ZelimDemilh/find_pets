# Сайт для поиска животных
Сайт на котором люди могут взять себе питомца!

[FindPets](https://find-pets-intocode.herokuapp.com/) - Ссылка на сайт

---

+ Авторизация
+ Просмотр категорий питомцев
+ Поиск питомцев по именам!
+ Просмотр полной информации про питомца
+ Возможность добавления на сайт своего объявления
+ Просмотр своих объявлений


![JUST-GIF](/first.gif)

+ При авторизации есть возможность добавлять новые объявления, а также удалять их.

![JUST-GIF](/second-gif.gif)

## Запуск проекта

---

Для запуска проекта вам необходимо:

Клонировать проект на ваше устройство командой:
```
$git clone https://github.com/shaulaev/find_pets
```
Указать путь к проекту:
```
cd find_pets
```
Установить зависимости при помощи:
```
npm i
```
Указать путь к клиенту и там тоже установить зависимости
```
cd client
npm i
```
И запустить проект при помощи команды: 
```
npm run dev
```
